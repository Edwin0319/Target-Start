# Product Overview

A browser-based 2D platformer game with an integrated level editor. Players can create custom maps using a grid-based editor with various game elements (spawn points, stars, blocks, jump springs, sloped blocks, and moving platforms), then play through their creations. The game supports multiple levels and includes both demo and challenge modes.

## Key Features

- Visual map editor with real-time preview
- Multiple game elements with placement validation
- 3 levels per game session
- Map import/export functionality
- Grid-based level design (24x13 cells)
